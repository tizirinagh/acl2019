While the relations in the expert-annotated dialogue are color-coded according to their type (relation-type appears on hover in the graphs of the games listed above), the relations in the "LF-generated" graph are color-coded according to whether the LF correctly or incorrectly attached the dialogue units:

  • Blue == true positive attachments
  • Magenta == false positive attachments
  • Orange == false negative attachments, i.e. the units should have been attached

*Note that we ignored for this study dialogues which contain only non-linguistic dialogue units. These still appear in the game graphs above, and should be disregarded.
